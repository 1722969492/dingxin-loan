# 鼎信助贷官网

> 专业贷款服务公司官网 — 让融资更简单

## 项目概览

一个现代、响应式的助贷公司品牌官网，采用纯前端技术构建，适合部署在任何静态托管平台。

## 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 自定义属性、Grid/Flexbox、Blur 毛玻璃、渐变、响应式
- **JavaScript** — 纯原生，无外部依赖

## 页面内容

| 版块 | 内容 |
|------|------|
| Hero | 品牌标语 + 贷款计算器（实时）|
| 合作伙伴 | 10家合作银行展示 |
| 关于我们 | 公司介绍 + 四大特色 |
| 服务项目 | 6大贷款产品卡片 |
| 核心优势 | 4大亮点数据化呈现 |
| 办理流程 | 四步流程引导 |
| 成功案例 | 3个真实案例 |
| FAQ | 6条高频问答（可折叠）|
| 联系我们 | 联系方式 + 在线表单 |

## 部署方式

直接部署 `index.html` 到任意静态服务器即可。

### 本地预览

用浏览器直接打开 `index.html` 即可。

### 生产部署

推荐使用 Nginx、Apache、或腾讯云 COS / 阿里云 OSS 静态托管。

```
/index.html
/README.md
```

## 定制指南

修改 `:root` 中的 CSS 变量即可更换主题色：

```css
:root {
  --primary: #1a56db;    /* 主色调 */
  --accent: #f59e0b;     /* 强调色 */
  --bg-dark: #0f172a;    /* 深色背景 */
}
```

## 注意

- 联系方式、地址、成功案例为样板内容，上线前需替换为真实信息
- 表单提交当前为控制台输出，需接入后端 API
- 网站备案号需替换为实际备案信息
